# AI Czochralski Landing Page

Современный лендинг на тему **«ИИ-оптимизация выращивания монокристаллического кремния методом Чохральского»**.

## 🚀 Быстрый старт

### Локальный запуск

1. Откройте файл `index.html` в любом современном браузере:
   ```bash
   # macOS
   open index.html
   
   # Linux
   xdg-open index.html
   
   # Windows
   start index.html
   ```

2. Или используйте локальный сервер (рекомендуется):
   ```bash
   # С помощью Python 3
   python -m http.server 8000
   
   # Затем откройте http://localhost:8000
   ```

## 🎨 Особенности дизайна

- **Тёмная тема** с акцентными цветами (синий #3b82f6, зелёный #10b981)
- **Адаптивная вёрстка** (mobile-first, 320px – 1920px)
- **Плавные анимации** при скролле (Intersection Observer API)

## 🛠️ Расширение проекта

### Добавление новых секций

1. Создайте HTML-разметку в `index.html`
2. Добавьте стили в `css/styles.css` с использованием BEM
3. При необходимости добавьте JS-функциональность в `js/main.js`

### Изменение цветовой схемы

Отредактируйте CSS-переменные в `:root`:

```css
:root {
  --color-accent-blue: #3b82f6;  /* Основной акцент */
  --color-accent-green: #10b981; /* Дополнительный акцент */
}
```

### Обновление команды

Измените данные в секции `<section class="team">` в `index.html`.

Для экспорта данных команды в консоль выполните:
```javascript
exportTeamData(); // В консоли браузера
```
